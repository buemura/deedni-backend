# deedni

deedin is a job board inspired in `indeed`

## Setup

1.  Install dependencies

    ```bash
     npm instal
    ```

2.  Setup prisma

    ```bash
    npx prisma generate
    ```

3.  Run application

    ```bash
    npm run start:dev
    ```

4.  Access swagger documentation on browser \
    http://localhost:8080/api/docs
